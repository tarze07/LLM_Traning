# AGENTS.md

Pracujesz w repozytorium korzystającym ze środowiska warsztatowego agenta (agent workbench).

Zapoznaj się z poniższymi plikami przed przystąpieniem do pracy:

1. `agent_state.json` — stan końcowy ostatniej sesji.
2. `task_board.json` — tablica zadań: aktualne zadania oraz planowane kroki.
3. `docs/agent-rules.md` — zasady pracy agenta: inicjalizacja, strefy zakazane, kryteria ukończenia, obsługa niepewności i zatwierdzanie zmian.
4. `docs/reliability-policy.md` — polityka niezawodności: obsługa typowych błędów, do których eliminacji zaprojektowano to środowisko.
5. `docs/handoff-protocol.md` — protokół przekazania: wymagane informacje na koniec sesji.
6. `docs/reviewer-rubric.md` — kryteria oceny wykonanej pracy.

Polecenie weryfikacji: sprawdź pole `acceptance_criteria` w aktywnym zadaniu na tablicy.

Wersja pakietu: 1.0.0
