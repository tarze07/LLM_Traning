# Polityka niezawodności

Stół warsztatowy obsługuje pięć powtarzających się w branży trybów awarii:

1. Działanie halucynacyjne — złapane przez zestaw reguł + bramka weryfikacyjna.
2. Przesunięcie zakresu — wykryte podczas sprawdzania różnic w umowie zakresu.
3. Błędy kaskadowe — wyłapane przez rekordy zwrotne + odmowa przy zerowym wyjściu.
4. Utrata kontekstu — pochłaniana przez pamięć repo; rozmowa nie jest źródłem prawdy.
5. Niewłaściwe użycie narzędzia — wykryte przez wymiar weryfikacji rubryki recenzenta.

Polityka jest egzekwowana przez bramkę weryfikacyjną. Ścieżka zastąpienia jest podpisana
i audytowane; agenci nie mogą się samodzielnie obejść.
