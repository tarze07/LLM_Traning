# Rubryka recenzenta

Pięć wymiarów, punktacja od 0 do 2.

1. Dopasowanie problemu — czy zmiana rozwiązała zadanie zgodnie z opisem?
2. Dyscyplina zakresu – czy zmiany ograniczały się do umowy?
3. Założenia – czy ukryte założenia są spisane?
4. Jakość weryfikacji – czy akceptacja rzeczywiście dowodzi celu?
5. Gotowość do przekazania — czy następna sesja może rozpocząć się gładko?

Razem >= 7 bez zer: zaliczony. Razem 5-6: miękka porażka. Poniżej 5 lub dowolnego zera: twardy błąd.
