# AGENTÓW.md

Pracujesz w repozytorium działającym ze środowiskiem roboczym agenta.

Przeczytaj je, zanim zaczniesz działać:

1. `agent_state.json` — miejsce, w którym zakończyła się ostatnia sesja.
2. `task_board.json` — co leci, co dalej.
3. `docs/agent-rules.md` — uruchomienie, zabronione, zrobione, niepewność, zatwierdzenie.
4. `docs/reliability-policy.md` — tryby awarii, do których absorpcji zaprojektowano ten stół warsztatowy.
5. `docs/handoff-protocol.md` — co musi dać koniec sesji.
6. `docs/reviewer-rubric.md` — sposób oceny wykonanej pracy.

Polecenie weryfikacji: patrz `acceptance_criteria` w aktywnym zadaniu na tablicy.

Wersja pakietu: 1.0.0
